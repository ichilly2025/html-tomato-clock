# FocusFlow 番茄钟应用

一个简洁美观的番茄钟应用，帮助您通过25分钟专注和5分钟休息的循环提高工作效率。

## 功能特性

- 25分钟工作模式和5分钟休息模式
- 直观的倒计时显示
- 开始、暂停、重置控制
- 一键切换工作/休息模式
- 计时结束提醒对话框
- 深色主题配合深橙色主题色

## 运行要求

- Flutter SDK 3.0.0 或更高版本
- Dart SDK 3.0.0 或更高版本

## 运行方法

1. 确保已安装Flutter开发环境
2. 克隆或下载项目代码
3. 在项目目录中运行：
   ```bash
   flutter pub get
   flutter run
   ```

## 项目结构

- `lib/main.dart` - 主应用文件，包含所有核心逻辑
- `pubspec.yaml` - 项目依赖配置

## 设计说明

- 采用Material Design 3设计规范
- 深色主题模式，减少视觉疲劳
- 深橙色主题色，提升视觉吸引力
- 响应式布局，适配不同屏幕尺寸

## 技术特点

- 使用Flutter原生setState进行状态管理
- 无复杂第三方依赖
- 代码简洁，注释清晰
- 符合Dart最佳实践
